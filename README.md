# java-web-dev-techjobs-persistent

### Description
A job searching/adding application that uses data added by the user and stored persistently a MySql database that searches for or adds jobs in the computer programming field.

### Things it can do

1. Allow a user to add jobs to the database.
2. Allow a user to view a list of all jobs stored in the database at once or view view a list of jobs by category/skill required.
3. Allow a user to view a job details page for each job stored in the database by selecting it.
4. Allow a user to search for jobs that are stored in the database:
     - By hiring company name.
     - By required skill.
     - Or any field that matches the search query.
